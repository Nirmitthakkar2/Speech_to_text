## üöÄ Step-by-Step Instructions

### Step 1: Get Your API Key from OpenRouter

1. **Open your web browser** (Chrome, Firefox, Safari, etc.)

2. **Go to this website:** https://openrouter.ai

3. **Create an account:**
   - Click "Sign Up" if you don't have an account
   - Or click "Log In" if you already have one
   - Follow the simple registration process

4. **Navigate to API Keys:**
   - After logging in, click on your name in the top-right corner
   - Select "Settings"
   - Click on "API Keys" in the left sidebar

5. **Create a new key:**
   - Click the "Create Key" button
   - You'll see a long code that looks like: `sk-or-v1-xxxxxxxxxxxxx`
   - Click the "Copy" button next to it

**Important:** Keep this code safe! Don't share it with anyone.

**Cost:** This is completely FREE! You don't need to pay anything.

---

### Step 2: Add the API Key to Your Application

1. **Open your Speech to Text application** in your browser

2. **Find the Settings button:**
   - Look at the top-right corner of the page
   - You'll see a gear icon (‚öôÔ∏è) labeled "Settings"
   - Click on it

3. **Enter your API key:**
   - You'll see a page titled "OpenRouter API Key"
   - Find the input box labeled "API Key"
   - Click inside the box
   - Paste your copied API key (Right-click ‚Üí Paste, or press Ctrl+V on Windows, Cmd+V on Mac)

4. **Save your key:**
   - Click the blue "Save API Key" button
   - You'll see a success message: "API Key saved successfully! ‚úì"

**Congratulations! You're done!** üéâ

---

### Step 3: Test Your API Key (Optional but Recommended)

1. **Stay on the Settings page**

2. **Click the "Test API Key" button**

3. **Check the result:**
   - If successful: You'll see "‚úì API Key is working!"
   - If there's an error: You'll see an error message explaining what went wrong

4. **If the test fails:**
   - Make sure you copied the entire API key
   - Check that there are no extra spaces
   - Try creating a new API key on OpenRouter and use that instead

---

### Step 4: Start Using the Application

1. **Go back to the main page:**
   - Click the back arrow (‚Üê) in the top-left corner
   - Or click "Speech to Text" in the header

2. **Start recording:**

   - Click stop when done

## üîí Security & Privacy

### Your API Key is Safe Because:

1. **Stored locally:** Your API key is saved only in your browser on your computer
2. **Not sent to our servers:** We never receive or store your API key
3. **Only you can access it:** Only you can see and modify your API key
4. **Easy to remove:** You can delete it anytime using the "Remove Key" button

**In simple terms:** It's like saving a password in your browser - it stays on your device only.

## ‚ùì Frequently Asked Questions

### Q1: Do I need to pay for the API key?
**A:** No! OpenRouter provides free API keys, and the AI model we use is completely free.

### Q2: What if I forget my API key?
**A:** No problem! Just go back to OpenRouter, create a new API key, and add it to the Settings page.

### Q3: Can I share my API key with others?
**A:** No! Your API key is like a password. Keep it private and don't share it with anyone.

### Q4: Do I need to enter the API key every time I use the app?
**A:** No! Once you save it, it stays saved in your browser. You only need to do this once.

### Q5: What if I use a different browser?
**A:** Each browser stores data separately. If you switch from Chrome to Firefox, you'll need to add the API key again in Firefox.

### Q6: How do I remove my API key?
**A:** Go to the Settings page and click the "Remove Key" button.

### Q7: What if the "Test API Key" button shows an error?
**A:** 
- Check your internet connection
- Make sure you copied the complete API key
- Try creating a new API key on OpenRouter
- Make sure there are no extra spaces before or after the key

### Q8: Will my API key expire?
**A:** OpenRouter API keys don't expire unless you delete them. Once set up, they work indefinitely.

**Solution:**
